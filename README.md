# Daily-tracker
Personal vibe coded project to track daily progress and plan the next day

the core idea of this project is the lack of a single app that can track everything about your day. It is a simple single page HTML document which saves data to a json in your own computer. I am someone who spends the whole day in the laptop and my brain is highly active it thinks of 100 tasks in 60 minutes. Most of which are interesting and important. I tend to jump between tasks so that I don't forget to do it later. I instead wanted to add it to a proper formatted list which can be easily accesible and help me stay focussed. 

The major features

-> Daily tasks planning
-> Tasks review log - accesible weekly monthly 
-> Habit Tracker
-> Declutter thoughts to write stuff and clear your mind

The html is connected to a json file writes and reads to it so you can pin it to your desktop when you open it and later even if gets crashed reopen and import the last accessed json. The json file autosaves after every entry.

After the first session export your json and next time open it, it is as simple as that. Okay track your day and enhance your productivity.
